# Интерактивный конструктор образовательных тестов

## 📋 Описание
Веб-приложение для создания и прохождения тестов по реляционным базам данных

## 🚀 Технологии
- **Frontend:** React 18, Redux Toolkit, React Router v6, CSS Modules
- **Backend:** Node.js, Express, Sequelize ORM
- **Database:** PostgreSQL 15
- **DevOps:** Docker, Docker Compose

## 📦 Установка

### Локальный запуск
```bash
# Backend
cd server
npm install
npm run dev

# Frontend (в другом терминале)
cd client
npm install
npm start
